# cite_mapper

Quick and dirty mapper of cite urns to standard abbreviations

## Installation

Add this line to your application's Gemfile:

    gem 'cite_mapper'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cite_mapper
